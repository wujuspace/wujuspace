5. Gradient Usage Guide

Usage Recommended Style Example
Logo / Hero visual Full multicolor gradient (Blue → Purple → Pink)
UI buttons / Icons Blue → Purple (for legibility)
Illustration (blobs) Purple → Pink or Blue → Cyan
Hover / highlights Accent gradient: Cyan → Lavender

Rule of Thumb

  • Use gradients for large organic shapes and backgrounds
  • Use flat accents for small UI details, icons, or highlights
  • Keep 1–2 gradients per section max — avoid clutter
  • When in doubt: gradient = hero element, accent = detail element

Typography Rules: Text Casing

Font Usage

  • Open Sans (Sans-serif): Used for body text, UI elements, navigation, buttons, forms, metadata, and all general content.
  • Prata (Serif): Used for headings (H1-H3), main headlines, hero text, titles, and display text that requires emphasis and elegance.

Text Casing Rules

Sentence Case (Default)

Use for: Body paragraphs, descriptions, labels, form inputs, buttons, navigation links, metadata, and most general content.

Example: "It felt like I belonged."
Example: "We had to leave Russia eight years ago with two suitcases and our baby..."
Title Case

Use for: Section headers, card titles, page titles, menu items, and UI component labels.

Example: "Open Sans / Prata"
Example: "Project Highlight"
Uppercase

Use for: Small labels, badges, tags, navigation indicators, and emphasis where appropriate. Use sparingly.

Example: "PROJECT HIGHLIGHT"
Example: "NEW"
Lowercase

Use for: Brand names, URLs, technical terms, and specific stylistic choices (e.g., "wuju" as brand name).

Example: "wuju"
Example: "development"

Text Styling: Emphasis & Decoration

Bold (strong or b)

Use for: Important keywords, headings within body text, call-to-action emphasis, key terms, and UI labels that need emphasis.

Example: "We had to leave Russia eight years ago with two suitcases and our baby..."

Italic (em or i)

Use for: Emphasis, quotes, technical terms, foreign words, book/movie titles, and subtle emphasis in body text.

Example: "It felt like I belonged. We had to leave Russia eight years ago..."

Underline (u)

Use for: Links (default browser style), important annotations, or specific emphasis. Avoid for general text emphasis.

Example: "Text source: Humans of New York"

Strike-through (s or del)

Use for: Deleted/corrected text, showing changes, deprecated content, or showing original vs. updated information.

Example: "Original price: $100 Now $50"

Combined Styling

You can combine styles for additional emphasis, but use sparingly to maintain readability.

Example: "Important notice: Please read carefully."

Brand Voice Grade A

Voice Examples

❌ DON'T: "Click Here" - Generic, passive
